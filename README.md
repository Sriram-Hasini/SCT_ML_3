# Cats and Dogs Image Classification Using SVM

## Objective
To classify images of cats and dogs using a Support Vector Machine (SVM) algorithm.

## Tools Used
- Python
- Google Colab
- OpenCV
- NumPy
- Scikit-Learn
- Matplotlib

## Workflow
1. Load dataset
2. Preprocess images
3. Resize images to 32x32
4. Convert images into feature vectors
5. Normalize data
6. Split dataset into training and testing sets
7. Train SVM model
8. Evaluate model performance
9. Save trained model

## Results
- Accuracy: 48.75%
- Confusion Matrix Generated
- Classification Report Generated
- Model Saved Successfully

## Algorithm
Support Vector Machine (LinearSVC)
